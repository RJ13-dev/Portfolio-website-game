Full-stack · Django + TypeScript + Python

This portfolio & Stimulus.

A full-stack portfolio website built with Django and Django REST Framework that also acts as the backend for my Stimulus game. It serves this single-page site, exposes a JWT-secured game API, caches a leaderboard in Redis, persists data in PostgreSQL, and powers the Contact chatbot through the Gemini API (the key stays server-side). The whole app is containerised with Docker behind Nginx, documented with OpenAPI/Swagger, and shipped to AWS through a GitHub Actions CI/CD pipeline.

Stimulus is a warm, narrative puzzle game where every puzzle nudges you toward one small act of self-care. It was first built in Python — a downloadable Python version of the game still exists — and then rebuilt as a browser game using a custom TypeScript engine with HTML5 Canvas effects, wired to the Django backend so players can save their coins, progress, and the letters they have read.

Learning: Building the site and game together showed me how one Django backend can power very different frontends — a marketing site, an AI chatbot, and a real-time game — and how auth, caching, containers, and CI/CD fit together to ship something to production.

Learnings.

The lessons that shaped how I build software — drawn from my full-stack projects and sharpened by hands-on systems and support work.

Architecture

Design the whole system.

Building full-stack apps with Angular, Spring Boot, Node.js, Redis, MongoDB, and Kafka taught me to design for the entire journey of data — input, validation, API contracts, caching, and events — instead of just making one screen work in isolation.

Debugging

Reason from the symptoms.

Resolving real incidents sharpened the way I debug code: reproduce the issue, isolate the cause from the noise, check logs and edge cases, then verify the fix. That same discipline is what keeps my software stable once it ships.

IT Support

Solve it for the user.

Level 1 and Level 2 helpdesk work across Microsoft 365, Active Directory, Entra ID, VPN, and Windows devices taught me to diagnose issues fast, prioritise against SLAs, escalate cleanly, and document every step — keeping users unblocked while staying calm under pressure.

APIs & data

Design clean contracts.

Working with RESTful APIs, JWT auth, Redis caching, and Kafka messaging taught me to design clear, well-documented interfaces between services — so systems stay decoupled, scalable, and easy for the next developer to build on.

Shipping

Get it to production.

Docker, GitHub Actions, and AWS taught me that writing the code is only half the job. I learned to containerise, automate testing and deployment, and actually run software in production instead of leaving it as a local demo.

Craft

Write code people can read.

Documentation, technical writing, and customer-facing work trained me to write for the next person — which carries straight into readable code, clear commits and pull requests, and APIs a teammate can pick up without a handover.
